# Textile Credit App

A simple credit / udhaar management app for tracking customers, invoices, and payments.

## Run locally (optional, needs Node.js installed)
```
npm install
npm run dev
```

## Deploy for free (no coding needed) — see full step-by-step guide from Claude.
Quick version:
1. Upload this folder to a new GitHub repository (via github.com's "Add file → Upload files", no git commands needed).
2. Go to vercel.com → sign in with GitHub → "Add New Project" → import the repo → Deploy.
3. Vercel gives you a live URL (e.g. `textile-credit.vercel.app`) — open it on your phone and use "Add to Home Screen".

## Notes
- Data is stored in the browser's local storage on whichever device/browser you use the app on. It does not sync between devices automatically.
